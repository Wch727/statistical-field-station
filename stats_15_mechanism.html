§XV

机理建模与微分方程

从第一性原理推导系统方程。差分方程、常微分方程、Logistic 增长、SIR 传染病模型、Lotka-Volterra 捕食者-猎物、扩散方程、数值求解 (solve_ivp)、参数估计 (curve_fit)。回答「系统如何随时间演化？」

竞赛真题场景

Q：美赛 MCM A 题：鱼类种群如何响应捕捞压力？→ Logistic 增长 + 捕捞项。
Q：国赛/美赛：传染病如何传播？何时达到峰值？→ SIR 模型 + 参数估计。
Q：美赛 ICM：物种竞争如何影响种群动态？→ Lotka-Volterra 竞争模型。
Q：物理/工程类：热传导、扩散过程中的温度分布？→ 扩散方程 + 数值求解。
→ 机理建模的关键不是解方程（Python 会帮你解），而是把自然规律翻译成微分/差分关系。

15.1 核心概念：从机理到方程

机理建模 vs 数据建模

数据建模（§VI 回归、§VIII 分类）：从数据出发，寻找输入-输出之间的统计关系。模型是「黑箱」或「灰箱」——你不需要知道为什么 X 影响 Y，只需要知道 X 能预测 Y。

机理建模（本章）：从第一性原理（物理定律、化学反应、生物学规律）出发，推导出描述系统行为的数学方程。模型是「白箱」——方程中的每个符号都有明确的物理/生物/化学含义。

机理建模的通用框架

Step 1 — 守恒关系：系统中什么量守恒？质量、能量、动量、种群数量……守恒关系通常写成：

\[ \text{变化率} = \text{流入速率} - \text{流出速率} + \text{生成速率} - \text{消耗速率} \]

Step 2 — 本构关系：流入、流出、生成、消耗如何依赖于系统状态？例如：Fick 扩散定律（流量 ∝ 浓度梯度）、Newton 冷却定律（热流 ∝ 温差）、Michaelis-Menten 酶动力学。

Step 3 — 闭合：方程数 = 未知数个数。如果不够，添加假设（如稳态假设、平衡假设）或经验关系。

15.2 差分方程

离散时间的演化

当时间以离散步长（天、月、年、代）推进时，系统状态由差分方程描述。差分方程是微分方程的离散版本，也是竞赛中更易处理的建模方式。

一阶线性差分方程

\[ x_{t+1} = a x_t + b \]

通解：如果 \(a \neq 1\)：

\[ x_t = a^t x_0 + b \cdot \frac{1 - a^t}{1 - a} \]

稳态：令 \(x_{t+1} = x_t = x^*\)，得 \(x^* = \frac{b}{1-a}\)。稳态稳定当且仅当 \(|a| < 1\)。

竞赛应用：人口模型（逐年迭代）、经济增长（GDP 年增长率）、药物代谢（每半衰期残留量减半）。

Logistic 差分方程（离散 Logistic）

\[ x_{t+1} = r x_t (1 - x_t) \]

这是著名的混沌模型——当 \(r\) 增大时，系统从稳定点 → 周期振荡 → 混沌。竞赛中注意：连续 Logistic 方程（ODE）不会混沌，只有离散版本才会。这提醒我们，离散化可能引入原连续模型没有的行为。

15.3 常微分方程基础

连续时间的演化

常微分方程（Ordinary Differential Equation, ODE）描述系统状态随连续时间的变化率。竞赛中遇到的 ODE 绝大多数是一阶系统——变化率只依赖于当前状态。

一阶 ODE 的标准形式

\[ \frac{dy}{dt} = f(t, y), \quad y(t_0) = y_0 \]

解析解 vs 数值解：只有少数 ODE 有解析解（如线性 ODE、可分离变量 ODE）。竞赛中绝大多数 ODE 需要用数值方法求解——Python 的 scipy.integrate.solve_ivp 是标准工具。

稳定性分析：对于自治系统 \(\frac{dy}{dt} = f(y)\)，平衡点 \(y^*\) 满足 \(f(y^*) = 0\)。如果 \(f'(y^*) < 0\)，平衡点稳定（扰动后系统回到平衡点）；如果 \(f'(y^*) > 0\)，平衡点不稳定。

15.4 经典模型

15.4.1 指数增长与衰减

最基本的动力学模型

\[ \frac{dN}{dt} = rN \quad \Rightarrow \quad N(t) = N_0 e^{rt} \]

\(r > 0\) → 指数增长（细菌繁殖、复利）；\(r < 0\) → 指数衰减（放射性衰变、药物代谢）。翻倍时间：\(T_2 = \ln 2 / r\)；半衰期：\(T_{1/2} = \ln 2 / |r|\)。

局限：指数增长不可持续——资源有限，增长率最终会下降。这引出了 Logistic 模型。

15.4.2 Logistic 增长

有限资源下的增长——S 曲线

\[ \frac{dN}{dt} = rN\left(1 - \frac{N}{K}\right) \]

其中 \(K\) 是环境承载力（carrying capacity）。当 \(N \ll K\) 时，增长近似指数；当 \(N \to K\) 时，增长率趋近于零。

解析解：

\[ N(t) = \frac{K}{1 + \left(\frac{K - N_0}{N_0}\right) e^{-rt}} \]

拐点：在 \(N = K/2\) 处，增长率达到最大。拐点时间：\(t_{\text{inf}} = \frac{1}{r} \ln\left(\frac{K - N_0}{N_0}\right)\)。

15.4.3 SIR 传染病模型

Kermack-McKendrick 模型 (1927)

将人群分为三类：Susceptible（易感者）、Infectious（感染者）、Recovered（康复者/移除者）。

\[ \begin{aligned} \frac{dS}{dt} &= -\beta \frac{SI}{N} \\ \frac{dI}{dt} &= \beta \frac{SI}{N} - \gamma I \\ \frac{dR}{dt} &= \gamma I \end{aligned} \]

其中 \(\beta\) = 传播率（单位时间一个感染者传染的人数），\(\gamma\) = 恢复率（\(1/\gamma\) = 平均感染期），\(N = S + I + R\) = 总人口。

基本再生数：\(R_0 = \beta / \gamma\)——一个感染者在完全易感人群中平均传染的人数。注意：\(R_0 > 1\) 是疫情扩散的必要条件；初始阶段是否增长取决于有效再生数 \(R_{\text{eff}}(0) = R_0 \cdot S_0/N\)——只有当 \(R_{\text{eff}}(0) > 1\) 即 \(S_0 > N/R_0\) 时，感染人数才会在初期增长。

峰值条件：令 \(dI/dt = \beta SI/N - \gamma I = 0\)，得 \(S = \gamma N / \beta = N / R_0\) 时 \(I\) 达到最大值。若使用比例 \(s = S/N\)，则峰值条件简洁地写为 \(s = 1 / R_0\)。最终规模由超越方程 \(S_\infty = S_0 \exp\left(-\frac{\beta}{\gamma N}(N - S_\infty)\right)\)（人数形式）或 \(s_\infty = s_0 \exp\left(-R_0(1 - s_\infty)\right)\)（比例形式）决定。

15.4.4 捕食者-猎物模型 (Lotka-Volterra)

两个物种的相互作用

\[ \begin{aligned} \frac{dH}{dt} &= \alpha H - \beta H P \quad \text{（猎物 } H \text{：自然增长 − 被捕食）} \\ \frac{dP}{dt} &= \delta H P - \gamma P \quad \text{（捕食者 } P \text{：捕食收益 − 自然死亡）} \end{aligned} \]

特性：① 解是周期性的——猎物和捕食者数量交替振荡，相位差约 1/4 周期。② 存在一个非平凡平衡点：\(H^* = \gamma / \delta\)，\(P^* = \alpha / \beta\)。③ 平衡点是中心（中性稳定）——扰动后系统在新的轨道上振荡，不回到原平衡点。

竞赛变体：① 竞争模型（两个物种竞争同一资源）② 互惠模型（两个物种互相帮助）③ 加入 Allee 效应（种群过小时增长率下降）。

15.4.5 扩散与热传导

空间分布的演化

当物质或热量在空间中扩散时，浓度 \(u(x,t)\) 满足扩散方程（Fick 第二定律）：

\[ \frac{\partial u}{\partial t} = D \frac{\partial^2 u}{\partial x^2} \]

其中 \(D\) 是扩散系数。这是一维热传导方程也是同一形式（Fourier 定律）。

竞赛简化：偏微分方程（PDE）的数值求解较复杂。竞赛中通常采用：① 离散化——将空间分成网格，每个格点上的浓度用 ODE 描述（方法线）。② 稳态近似——令 \(\partial u/\partial t = 0\)，退化为 Laplace 方程 \(\nabla^2 u = 0\)。③ 对称性简化——利用球对称或柱对称降维。

15.5 数值求解

Python 工具：scipy.integrate.solve_ivp

solve_ivp 是 SciPy 求解初值问题（Initial Value Problem）的标准函数。支持多种算法：

方法适用场景精度
RK45（默认）非刚性 ODE 首选中-高
RK23低精度快速求解低-中
DOP853高精度需求高
Radau / BDF刚性 ODE（如化学反应）中-高

基本用法：

from scipy.integrate import solve_ivp
def sir(t, y, beta, gamma, N):
    S, I, R = y
    return [-beta*S*I/N, beta*S*I/N - gamma*I, gamma*I]
sol = solve_ivp(sir, [0, 100], [S0, I0, R0], args=(0.3, 0.1, 1000),
                dense_output=True, max_step=1.0)
竞赛技巧：① 如果 ODE 求解极慢，检查是否是刚性问题——改用 Radau 或 BDF。② 设置 max_step 防止求解器跳过重要细节（如 SIR 的峰值）。③ 用 dense_output=True 可以在任意时间点插值得到解。④ 对于复杂 ODE 系统，先画相图（phase portrait）理解定性行为，再调参数。

15.6 参数估计

从数据反推模型参数

机理模型中的参数（如 SIR 的 β、γ）通常不能直接测量，需要从数据中估计。这是竞赛中机理建模题的标配环节。

方法一：最小二乘拟合（curve_fit）：

\[ \hat{\boldsymbol{\theta}} = \arg\min_{\boldsymbol{\theta}} \sum_{i=1}^{n} (y_i^{\text{obs}} - y(t_i; \boldsymbol{\theta}))^2 \]

方法二：最大似然估计（MLE）：当数据有明确的概率模型时（如计数数据→Poisson），MLE 比最小二乘更准确。

方法三：贝叶斯推断（MCMC）：当需要参数的不确定性量化时，MCMC 给出参数的后验分布。

竞赛流程：① 先手动调参，让模型趋势大致匹配数据。② 用 curve_fit 或 minimize 精调。③ 报告参数估计值 ± 标准误。④ 画拟合图（数据点 + 模型曲线）。

参数估计的常见陷阱：① 参数不可识别——多组参数产生相同的拟合曲线（如 SIR 中只有 R₀=β/γ 可识别，β 和 γ 单独不可识别）。解决：固定一个参数或使用先验信息。② 过拟合——参数太多而数据太少。附加正则化或使用更简单的模型。③ 局部最优——curve_fit 可能陷入局部极小。解决：多组初始值、全局优化算法（differential_evolution）。

15.7 竞赛实战流程

机理建模题竞赛标准流程

Step 1：识别核心机制。什么在变化？变化的原因是什么？守恒关系是什么？→ 写出文字描述。
Step 2：建立数学模型。将文字描述翻译为微分方程或差分方程。明确每个符号的含义和单位。
Step 3：分析模型。求平衡点、稳定性分析、相图。这些定性分析在论文中非常重要。
Step 4：数值求解。Python solve_ivp 求解 ODE 系统。画出时间序列图和相图。
Step 5：参数估计。如果有数据，用 curve_fit 估计参数。报告参数值和置信区间。
Step 6：模型验证。留出部分数据验证模型预测。报告预测误差。
Step 7：灵敏度分析。改变关键参数 ±20%，观察模型输出的变化。

竞赛加分项：① 相图分析——在 (S, I) 平面上画出轨道，展示系统动力学。② 分岔分析——当参数变化时，系统行为如何定性改变（如从稳定到振荡）。③ 模型对比——比较不同模型（如 SIR vs SEIR）的拟合效果，用 AIC 选择。④ 不确定性量化——参数的后验分布或 Bootstrap 置信区间。⑤ 模型扩展——在基本模型上添加符合实际的项（如 SIR 加疫苗接种、Logistic 加 Allee 效应）。
