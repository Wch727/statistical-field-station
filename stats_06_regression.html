§VI

线性回归

OLS 矩阵推导、多元回归、交互项与多项式、多重共线性 (VIF)、变量选择 (LASSO/Ridge)、WLS（加权最小二乘）、分位数回归、残差诊断、AIC/BIC。

竞赛真题场景

Q：某年全国大学生数学建模竞赛 B 题：建立回归模型预测城市综合发展指数。需要从 20 个候选指标中选出最重要的 5-8 个，建立多元回归模型，并诊断模型假设是否满足。
Q：美赛 MCM 中经常要求建立预测模型——回归是首选。但评委期望看到的不只是「跑了一个回归」，而是完整的建模流程：变量选择 → 模型诊断 → 模型比较 → 预测与验证。
→ 回归是建模手最核心的工具。一个好的回归模型需要经过变量选择、共线性诊断、残差分析和模型验证四个步骤。

6.1 OLS 估计的矩阵推导

从 RSS 最小化到正规方程

对于多元线性回归 \(\mathbf{Y} = \mathbf{X}\boldsymbol{\beta} + \boldsymbol{\varepsilon}\)，最小二乘估计量是使残差平方和最小化的 β：

\[ \text{RSS}(\boldsymbol{\beta}) = (\mathbf{Y} - \mathbf{X}\boldsymbol{\beta})'(\mathbf{Y} - \mathbf{X}\boldsymbol{\beta}) \] \[ \frac{\partial \text{RSS}}{\partial \boldsymbol{\beta}} = -2\mathbf{X}'\mathbf{Y} + 2\mathbf{X}'\mathbf{X}\boldsymbol{\beta} = 0 \quad\Rightarrow\quad \hat{\boldsymbol{\beta}} = (\mathbf{X}'\mathbf{X})^{-1}\mathbf{X}'\mathbf{Y} \]

β̂ 的协方差矩阵：\(\text{Var}(\hat{\boldsymbol{\beta}}) = \sigma^2(\mathbf{X}'\mathbf{X})^{-1}\)。每个 β̂_j 的标准误：\(\text{SE}(\hat{\beta}_j) = \hat{\sigma}\sqrt{(\mathbf{X}'\mathbf{X})^{-1}_{jj}}\)。
Gauss-Markov 定理：在满足 LINE 假设的条件下，OLS 估计量是 BLUE (Best Linear Unbiased Estimator)。

\[ \hat{\boldsymbol{\beta}} = (\mathbf{X}'\mathbf{X})^{-1}\mathbf{X}'\mathbf{Y} \qquad \text{Var}(\hat{\boldsymbol{\beta}}) = \sigma^2(\mathbf{X}'\mathbf{X})^{-1} \qquad \hat{\sigma}^2 = \frac{\text{RSS}}{n-p} \] \[ \hat{\beta}_1 = \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sum (x_i - \bar{x})^2} = r_{xy} \cdot \frac{s_y}{s_x} \qquad \hat{\beta}_0 = \bar{y} - \hat{\beta}_1 \bar{x} \]

6.2 多元回归、交互项与多项式

从简单回归到建模

多元回归：\(Y = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \cdots + \beta_p X_p + \varepsilon\)。β_j 的含义是「在其他变量不变的情况下，X_j 每增加 1 单位，Y 平均变化 β_j 单位」。
交互项：\(Y = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \beta_3 (X_1 \cdot X_2) + \varepsilon\)。X₁ 的效应 = β₁ + β₃X₂，取决于 X₂ 的值。交互项显著 → X₁ 和 X₂ 的效应不是简单相加的。
多项式回归：\(Y = \beta_0 + \beta_1 X + \beta_2 X^2 + \cdots + \beta_p X^p + \varepsilon\)。虽然对 X 是非线性的，但对 β 仍然是线性的——仍可用 OLS。

\[ Y = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \beta_3 (X_1 X_2) + \varepsilon \qquad \frac{\partial Y}{\partial X_1} = \beta_1 + \beta_3 X_2 \]

6.3 多重共线性与 VIF

当 X 变量之间高度相关

当多个自变量高度相关时，X'X 接近奇异矩阵，(X'X)⁻¹ 的对角元素极大 → β̂ 的方差极大 → 系数估计非常不稳定。
VIF (Variance Inflation Factor)：\[ \text{VIF}_j = \frac{1}{1 - R_j^2} \]其中 R²_j 是将 X_j 对其他所有 X 做回归的决定系数。VIF_j = 1 表示无共线性；VIF_j > 5 需要关注；VIF_j > 10 表示严重共线性。
条件数 (Condition Number)：\[ \kappa = \sqrt{\frac{\lambda_{\max}}{\lambda_{\min}}} \]其中 λ 是 X'X 的特征值。κ > 30 表示严重共线性。
处理：① 删除高度相关的变量之一 ② 用 PCA 降维（主成分回归）③ 用 Ridge 回归（L2 正则化）④ 增大样本量。

\[ \text{VIF}_j = \frac{1}{1 - R_j^2} \qquad \text{Var}(\hat{\beta}_j) = \frac{\sigma^2}{\sum (x_{ij} - \bar{x}_j)^2} \cdot \text{VIF}_j \]

6.4 变量选择与正则化

LASSO (L1) vs Ridge (L2) vs Elastic Net

LASSO (L1 正则化)：\[ \min_{\beta} \sum (y_i - \mathbf{x}_i'\boldsymbol{\beta})^2 + \lambda \sum |\beta_j| \]将不重要的系数精确压缩到 0，实现自动变量选择。λ 通过交叉验证选择。
Ridge (L2 正则化)：\[ \hat{\boldsymbol{\beta}}_{\text{Ridge}} = (\mathbf{X}'\mathbf{X} + \lambda\mathbf{I})^{-1}\mathbf{X}'\mathbf{Y} \]将系数向 0 缩小但不精确到 0。适合处理共线性。
Elastic Net：结合 L1 和 L2 的优点——既做变量选择又处理共线性。

\[ \text{AIC} = n\ln(\text{RSS}/n) + 2p \qquad \text{BIC} = n\ln(\text{RSS}/n) + p\ln(n) \]
AIC vs BIC：AIC 倾向于选择更多变量（预测导向），BIC 对参数数量惩罚更重（解释导向）。竞赛中建议同时报告两者，如果 AIC 和 BIC 选出的最优模型不同，说明模型选择存在不确定性，需要进一步讨论。

6.5 残差诊断的完整流程

六步诊断法

1. 残差 vs 拟合值：检查线性假设和同方差。无模式 → 好；漏斗形 → 异方差；曲线 → 非线性。
2. Q-Q 图：检查残差正态性。点偏离对角线 → 非正态。
3. Cook's Distance：\[ D_i = \frac{e_i^2}{p \cdot \text{MSE}} \cdot \frac{h_{ii}}{(1 - h_{ii})^2} \]度量第 i 个观测对全部拟合值的影响。D_i > 4/n → 高影响点。
4. Leverage：\[ h_{ii} = [\mathbf{X}(\mathbf{X}'\mathbf{X})^{-1}\mathbf{X}']_{ii} \]杠杆值 > 2p/n → 高杠杆点。
5. Durbin-Watson：\[ \text{DW} = \frac{\sum (e_t - e_{t-1})^2}{\sum e_t^2} \]检验残差自相关。DW ≈ 2 → 无自相关。
6. Breusch-Pagan：将 e² 对 X 做回归。显著 → 拒绝同方差假设（存在异方差）。

\[ D_i = \frac{e_i^2}{p \cdot \text{MSE}} \cdot \frac{h_{ii}}{(1 - h_{ii})^2} \qquad \text{DW} = \frac{\sum_{t=2}^n (e_t - e_{t-1})^2}{\sum_{t=1}^n e_t^2} \]

6.6 Box-Cox 变换

当 LINEAR 假设不成立时

如果残差 vs 拟合值图显示明显非线性，可以考虑变换响应变量 Y：

\[ Y^{(\lambda)} = \begin{cases} \dfrac{Y^\lambda - 1}{\lambda} & \lambda \neq 0 \\[8pt] \ln Y & \lambda = 0 \end{cases} \qquad \hat{\lambda} = \arg\max_{\lambda} \left[-\frac{n}{2}\ln\frac{\text{RSS}(\lambda)}{n} + (\lambda-1)\sum \ln y_i\right] \]

6.6A 加权最小二乘 (WLS)

当异方差不可避免时

OLS 假设所有观测的误差方差相同（同方差性）。当这个假设不成立时（如收入越高的家庭消费波动越大），OLS 仍然是无偏的，但不再是 BLUE（最优线性无偏估计）。WLS 给每个观测分配一个权重，使得方差大的观测权重小，方差小的观测权重大。

\[ \hat{\beta}_{\text{WLS}} = \arg\min_{\beta} \sum_{i=1}^{n} w_i (y_i - x_i^T\beta)^2 = (X^T W X)^{-1} X^T W y \]

其中 \(W = \text{diag}(w_1, \ldots, w_n)\)，\(w_i = 1/\sigma_i^2\)（理想权重 = 误差方差的倒数）。实践中，\(\sigma_i^2\) 未知——通常先用 OLS 拟合，再用残差的绝对值或平方回归得到估计的方差，然后迭代（可行广义最小二乘，FGLS）。

竞赛技巧：如果 Breusch-Pagan 检验显著（存在异方差），报告 WLS 结果比 OLS 更可信。在论文中，先报告 BP 检验 → 确认异方差 → 使用 WLS → 报告权重是如何估计的。这展现了完整的诊断-修正流程。

6.6B 分位数回归

不只关心均值

OLS 只建模条件均值 \(\mathbb{E}[Y \mid X]\)。但竞赛中，我们可能关心尾部——例如，什么因素影响最高 10% 的房价？什么因素降低最低 5% 的考试成绩？分位数回归建模条件分位数 \(Q_\tau(Y \mid X)\)，对分布的任意位置都能给出洞察。

\[ \hat{\beta}_{\tau} = \arg\min_{\beta} \sum_{i=1}^{n} \rho_{\tau}(y_i - x_i^T\beta) \quad\text{其中}\quad \rho_{\tau}(u) = u(\tau - \mathbf{1}\{u < 0\}) \]

\(\rho_{\tau}\) 是「check function」——对正残差和负残差施加不对称的惩罚。当 \(\tau = 0.5\) 时，分位数回归退化为中位数回归（LAD，最小绝对偏差）。

推导：为什么分位数回归等价于非对称损失？

对于 \(\tau \in (0,1)\)，我们希望找到一个值 \(q\) 使得 \(\tau\) 比例的数据在 \(q\) 之下，\(1-\tau\) 比例在 \(q\) 之上。这等价于最小化：

\[ \mathbb{E}[\rho_{\tau}(Y - q)] = \tau \int_{q}^{\infty} (y-q) dF(y) + (1-\tau) \int_{-\infty}^{q} (q-y) dF(y) \]

对 \(q\) 求导并令为 0，得到 \(F(q) = \tau\)——即 \(q\) 是第 \(\tau\) 分位数。

竞赛技巧：分位数回归在竞赛中特别有用当：① 关注极端值（如风险分析中的 VaR）。② 自变量对因变量不同分位数的影响不同（如教育对低收入和高收入人群的影响不同）。③ 数据有严重离群值——分位数回归天然稳健，不需要删除离群值。

6.6C 稳健回归 (Robust Regression)

当异常值不能删除时

OLS 的损失函数是平方误差 \(\sum (y_i - \hat{y}_i)^2\)——一个远离回归线的点会产生巨大的平方误差，强力拉动回归线。如果这个点是真实的（而非记录错误），你不想删除它，但也不想让它主导模型——稳健回归通过降低大残差的权重来解决这个问题。

M-估计 (M-Estimation)

稳健回归的核心是替代平方损失为增长较慢的损失函数 \(\rho(e)\)：

\[ \hat{\beta} = \arg\min_{\beta} \sum_{i=1}^{n} \rho\left(\frac{y_i - \mathbf{x}_i^T\beta}{\hat{\sigma}}\right) \]

三种常见的 \(\rho\) 函数：

方法\(\rho(e)\) 行为特点
Huber|e| ≤ c 时平方，|e| > c 时线性对小残差像 OLS，对大残差像 MAE
Tukey 双权 (Bisquare)|e| > c 时权重降为 0完全忽略极端异常值，需要好的初始值
Hampel三段式：平方 → 线性 → 0最灵活，但参数最多

调谐常数 \(c\) 控制稳健性：\(c=1.345\)（Huber, 95% 效率于正态），\(c=4.685\)（Tukey, 95% 效率）。

竞赛技巧：如果 Cook's D 检测到多个高影响点，但它们是真实数据不能删除 → 用稳健回归。论文中：「OLS 回归的残差诊断显示存在若干高影响点 (Cook's D > 4/n)。由于这些点代表真实的市场现象而非记录错误，我们使用 Huber 稳健回归来降低其影响。稳健回归的标准误通过 MM 估计获得，在异方差和非正态下均一致。」

6.6D 分类自变量编码

如何把「城市」「性别」「血型」放进回归？

回归模型需要数值输入，但竞赛中经常遇到分类变量。编码方式的选择会影响系数的解释。

编码方式做法系数解释适用场景
Dummy (Indicator)k-1 个 0/1 变量，以一个类别为参照每个系数 = 该类别与参照类的差异有一个自然参照类（如「对照组」）
One-Hotk 个 0/1 变量（无截距项）每个系数 = 该类别的均值无自然参照类，但需去掉截距
Effect (Deviation)k-1 个变量，参照类编码为 -1每个系数 = 该类别与总均值的偏差ANOVA 风格的解释
Ordinal有序类别编码为整数 1, 2, 3...假设类别间距相等有序变量（如教育程度）
竞赛陷阱：① 把分类变量直接编码为 1, 2, 3 放进回归——这隐含假设了类别的间距相等且有序（如「北京=1, 上海=2, 广州=3」→ 模型认为上海-北京 = 广州-上海，这毫无意义）。② 忘记去掉一个类别——k 个类别 + 截距 → 完全共线性（哑变量陷阱）。③ 参照类只有很少的观测 → 系数估计不稳定。

6.6E 交互项的解释

「效果取决于……」——交互效应

当模型中包含交互项 \(Y = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \beta_3 (X_1 \cdot X_2)\) 时，\(X_1\) 的效应不再是 \(\beta_1\)，而是 \(\beta_1 + \beta_3 X_2\)——取决于 \(X_2\) 的值。

简单斜率分析 (Simple Slopes Analysis)

为了解释交互效应，通常在 \(X_2\) 的三个值上估计 \(X_1\) 的效应：

\[ \begin{aligned} \text{低 } X_2 \text{ (均值 - 1 SD)：} & \quad \text{效应} = \beta_1 + \beta_3(\bar{X}_2 - s_{X_2}) \\ \text{中 } X_2 \text{ (均值)：} & \quad \text{效应} = \beta_1 + \beta_3\bar{X}_2 \\ \text{高 } X_2 \text{ (均值 + 1 SD)：} & \quad \text{效应} = \beta_1 + \beta_3(\bar{X}_2 + s_{X_2}) \end{aligned} \]

然后检验每个简单斜率是否显著不为 0。这揭示了交互效应的具体模式——例如「教育对收入的效应在高能力人群中更强」。

竞赛技巧：交互效应是竞赛论文的加分项。论文中：「回归分析揭示了教育年限与社会资本的显著交互效应 (β₃=0.18, p=0.003)。简单斜率分析表明，对于低社会资本的个体（均值-1SD），教育年限每增加 1 年，收入增加 320 元；而对于高社会资本的个体（均值+1SD），教育年限每增加 1 年，收入增加 680 元。这说明社会资本放大了教育的回报。」——这种分析展示了深度理解。
