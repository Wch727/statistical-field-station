竞赛核心问题类型 → 对应章节

数学建模竞赛中，数据分析类问题通常围绕以下十一种提问方式展开

Q1
哪些因素有关？ → §III 相关分析：Pearson / Spearman / Kendall / 偏相关 / dCor
Q2
两组数据是否存在显著差异？ → §IV 假设检验：t 检验 / 卡方 / Mann-Whitney / Bootstrap / 置换检验
Q3
某因素对结果有多大影响？ → §V 方差分析 + §VI 线性回归：ANOVA / 事后检验 / OLS / 多元回归 / 效应量
Q4
数据中有什么规律？ → §I 描述性统计 + §II 可视化：均值/方差/偏度/峰度 / 直方图/箱线图/散点图
Q5
这个样本属于哪一类？ → §VII 逻辑回归 + §VIII 分类方法：决策树 / 随机森林 / SVM / KNN
Q6
数据能否划分亚类？ → §IX 聚类分析：K-Means / 层次聚类 / DBSCAN + §X 模型评估与降维
Q7
未来趋势如何预测？ → §XI 时间序列分析 + §XII 预测问题专论：ARIMA / GM(1,1) / 预测区间 / 多模型比较 / 数据泄漏防御
Q8
如何安排最优方案？ → §XIII 优化与决策：线性规划 / 整数规划 / 多目标优化 / 动态规划 / 背包 / 选址 / TSP
Q9
哪个方案/地区最好？ → §XIV 综合评价与排序：熵权法 / TOPSIS / AHP / PCA / 模糊综合评价 / 灰色关联分析
Q10
系统如何随时间演化？ → §XV 机理建模：差分/微分方程 / Logistic / SIR / 捕食者-猎物 / 数值求解
Q11
最短路径？关键节点？随机模拟？ → §XVI 图论与仿真：最短路径 / MST / 中心性 / 蒙特卡洛 / 排队 / 元胞自动机

§I
描述性统计
中心趋势、离散程度、分布形态 —— 数据分析的第一步。从 Bessel 校正的数学推导到 Box-Cox 变换，从异常值检测到正态性检验。数模论文的数据分析部分，第一段永远是描述性统计。
均值/中位数/众数方差/SD/SEM偏度/峰度Bessel校正Box-CoxQ-Q图Shapiro-WilkMAD
 核心

数据可视化

优化与决策
把自然语言翻译成数学模型——线性规划（单纯形/对偶/影子价格）、整数规划（分支定界）、0-1 规划与逻辑约束、非线性规划（KKT）、多目标优化（Pareto/TOPSIS）、动态规划（Bellman 方程）、背包/选址/TSP/VRP 经典模型。

综合评价与排序
