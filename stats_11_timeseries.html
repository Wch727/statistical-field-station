§XI

时间序列分析

当数据按时间顺序排列时，相邻观测之间不再独立——今天的气温与昨天相关，本月的销售额与上月相关。时间序列分析专门处理这种自相关结构，涵盖分解、平稳性检验、ARIMA 建模和预测。这是竞赛中处理经济、环境、交通等时序数据不可或缺的工具。

竞赛真题场景

Q：国赛常考：根据历史数据预测未来趋势——如城市人口增长、污染物浓度变化、交通流量预测。
Q：美赛 MCM A 题：利用时间序列分析海洋温度变化趋势，预测未来气候变化。
Q：华中赛/电工杯：电力负荷预测、股票价格走势分析——ARIMA + 季节分解是标准答案。
→ 竞赛中，时间序列分析的核心不只是预测，更是「趋势+季节性+随机波动」的分解与解释。

11.1 时间序列的组成

经典分解模型

时间序列 \(y_t\) 通常被认为由四个成分组成：

\[ y_t = T_t + S_t + C_t + R_t \]

其中 \(T_t\) = 趋势（长期方向），\(S_t\) = 季节性（固定周期的波动），\(C_t\) = 循环（非固定周期的波动，经济学中常与趋势合并），\(R_t\) = 残差（随机噪声）。

加法模型：\(y_t = T_t + S_t + R_t\)（波动的幅度不随趋势变化）
乘法模型：\(y_t = T_t \times S_t \times R_t\)（波动的幅度随趋势增大，取对数可转为加法）

11.1.1 移动平均平滑

消除噪声、提取趋势的最简单方法：

\[ \hat{T}_t = \frac{1}{2m+1} \sum_{j=-m}^{m} y_{t+j} \]

对于季节性数据，使用等权重的 \(m\) 期移动平均（\(m\) = 季节周期长度）。

11.2 平稳性

严格平稳 vs 弱平稳

严格平稳：任意时间平移后，联合分布不变。
弱平稳（协方差平稳，竞赛中默认的「平稳」）：

  1. 均值恒定：\(\mathbb{E}[y_t] = \mu\)，不随时间变化
  2. 方差恒定：\(\text{Var}(y_t) = \sigma^2\)，不随时间变化
  3. 自协方差只依赖于时间间隔：\(\text{Cov}(y_t, y_{t-k}) = \gamma_k\)，只与滞后 \(k\) 有关

11.2.1 单位根检验 (ADF Test)

Augmented Dickey-Fuller 检验是判断序列是否平稳的标准方法。回归模型：

\[ \Delta y_t = \alpha + \beta t + \gamma y_{t-1} + \sum_{i=1}^{p} \delta_i \Delta y_{t-i} + \varepsilon_t \]

原假设 \(H_0: \gamma = 0\)（存在单位根，非平稳）vs 备择假设 \(H_1: \gamma < 0\)（平稳）。如果 p-value < 0.05，拒绝 \(H_0\)，序列是平稳的。

推导：为什么单位根意味着非平稳？

考虑最简单的 AR(1) 过程：\(y_t = \phi y_{t-1} + \varepsilon_t\)，其中 \(\varepsilon_t \sim N(0,\sigma^2)\)。

迭代展开：\(y_t = \phi^t y_0 + \sum_{j=0}^{t-1} \phi^j \varepsilon_{t-j}\)。

当 \(|\phi| < 1\)：\(\phi^t \to 0\)，\(\text{Var}(y_t) = \frac{\sigma^2}{1-\phi^2}\) 有限且恒定 → 平稳。
当 \(\phi = 1\)（单位根）：\(y_t = y_0 + \sum_{j=1}^{t} \varepsilon_j\)，\(\text{Var}(y_t) = t\sigma^2 \to \infty\) → 非平稳。
当 \(|\phi| > 1\)：方差指数增长 → 爆炸性非平稳。

因此 ADF 检验中 \(\gamma = \phi - 1\)，检验 \(\gamma = 0\) 就是检验 \(\phi = 1\)（单位根）。

11.2.2 差分：使非平稳序列平稳

\[ \text{一阶差分：}\ \Delta y_t = y_t - y_{t-1} \] \[ \text{季节性差分：}\ \Delta_s y_t = y_t - y_{t-s} \quad (s = \text{季节周期}) \]

大多数经济时间序列经过一次差分后变为平稳，称为 I(1)（一阶单整）。ARIMA 中的「I」就是差分阶数。

11.3 自相关与偏自相关

11.3.1 自相关函数 (ACF)

\[ \rho_k = \frac{\text{Cov}(y_t, y_{t-k})}{\text{Var}(y_t)} = \frac{\sum_{t=k+1}^{n} (y_t - \bar{y})(y_{t-k} - \bar{y})}{\sum_{t=1}^{n} (y_t - \bar{y})^2} \]

ACF 衡量相隔 \(k\) 期的观测之间的相关性，包括直接和间接的传递效应。例如，\(y_t\) 与 \(y_{t-2}\) 的相关可能完全是通过 \(y_{t-1}\) 传递的。

11.3.2 偏自相关函数 (PACF)

PACF 衡量相隔 \(k\) 期的观测之间的直接相关性，剔除了中间期的影响。PACF 是 AR(p) 模型阶数选择的关键工具。

ACF/PACF 快速识别法则：
AR(p) 过程：ACF 呈指数衰减或正弦波衰减（拖尾），PACF 在滞后 \(p\) 后截尾（变为 0）。
MA(q) 过程：ACF 在滞后 \(q\) 后截尾，PACF 呈指数衰减（拖尾）。
ARMA(p,q)：ACF 和 PACF 都拖尾。
这是 Box-Jenkins 建模方法的核心——通过 ACF/PACF 图选 p 和 q。

11.4 ARIMA 模型

11.4.1 AR(p)：自回归模型

\[ y_t = c + \phi_1 y_{t-1} + \phi_2 y_{t-2} + \cdots + \phi_p y_{t-p} + \varepsilon_t, \quad \varepsilon_t \sim N(0,\sigma^2) \]

AR(1) 是最简单的形式：\(y_t = c + \phi y_{t-1} + \varepsilon_t\)。\(|\phi| < 1\) 时平稳，\(\phi\) 接近 1 时有强持续性。

11.4.2 MA(q)：移动平均模型

\[ y_t = \mu + \varepsilon_t + \theta_1 \varepsilon_{t-1} + \theta_2 \varepsilon_{t-2} + \cdots + \theta_q \varepsilon_{t-q} \]

MA(q) 总是平稳的。MA(1) 模型 \(y_t = \mu + \varepsilon_t + \theta \varepsilon_{t-1}\) 中，\(\theta\) 捕捉了冲击的持续影响。

11.4.3 ARIMA(p,d,q) 完整模型

ARIMA 的三步构建

I(d) — 差分：对原始序列做 \(d\) 次差分使其平稳。\(d=0\) 时 ARIMA 退化为 ARMA。
AR(p) — 自回归：用过去 \(p\) 期的观测值预测当前值。
MA(q) — 移动平均：用过去 \(q\) 期的预测误差修正当前预测。

ARIMA(1,1,1) 的完整形式：

\[ \underbrace{(1 - \phi_1 B)}_{\text{AR(1)}} \underbrace{(1-B)}_{\text{I(1)}} y_t = c + \underbrace{(1 + \theta_1 B)}_{\text{MA(1)}} \varepsilon_t \]

其中 \(B\) 是后移算子：\(B y_t = y_{t-1}\)。

11.4.4 SARIMA：季节性 ARIMA

当数据有季节性模式时（如每 12 个月的周期性），SARIMA(p,d,q)(P,D,Q)ₛ 同时建模非季节性和季节性成分：

\[ \Phi_P(B^s)\ \phi_p(B)\ (1-B)^d\ (1-B^s)^D\ y_t = \Theta_Q(B^s)\ \theta_q(B)\ \varepsilon_t \]

其中 \(s\) 是季节周期（如月度数据 \(s=12\)，季度数据 \(s=4\)），\(P,D,Q\) 是季节性 AR/I/MA 的阶数。

11.4.5 模型选择：AIC vs BIC

准则公式特点
AIC\(\text{AIC} = -2\ln L + 2k\)倾向于选择更复杂的模型（\(2k\) 惩罚轻）
AICc\(\text{AICc} = \text{AIC} + \frac{2k(k+1)}{n-k-1}\)小样本修正，竞赛推荐
BIC\(\text{BIC} = -2\ln L + k\ln n\)倾向于选择更简单的模型（\(\ln n\) 惩罚重）

竞赛工具箱：ARIMA 建模流程

Step 1：画时序图，判断趋势和季节性 → 选择是否需要差分和季节差分。
Step 2：ADF 检验 → 确定差分阶数 \(d\)（通常 \(d=0\) 或 \(d=1\)）。
Step 3：画 ACF/PACF 图 → 初步确定 \(p\) 和 \(q\) 的范围。
Step 4：网格搜索 \(p,q \in \{0,1,2,3\}\)，用 AICc 选择最优模型。
Step 5：残差诊断——残差应为白噪声（Ljung-Box 检验 p > 0.05）。
Step 6：预测 + 预测区间（95% 置信带）。
竞赛技巧：如果数据有明显的季节性，优先考虑 SARIMA 而非简单 ARIMA。另外，对数变换可以稳定方差（乘法模型 → 加法模型）。

常见陷阱：① 不检验平稳性就直接建模——非平稳序列的 ACF 衰减极慢，会导致虚假回归。② 过度差分——差分次数超过必要，会引入不必要的 MA 结构，增加模型复杂度。③ 不检查残差——残差若有自相关，说明模型未充分捕捉数据结构。④ 在竞赛中报告 R² 而非 AICc——时间序列的 R² 通常很高（因为趋势），不代表模型好。

11.5 指数平滑与 ETS

ARIMA 的替代方案——指数平滑系列模型（ETS = Error-Trend-Seasonality）：

模型适用场景趋势季节性
简单指数平滑 (SES)无趋势、无季节无无
Holt 线性趋势有线性趋势加法无
Holt-Winters 加法趋势 + 加法季节加法加法
Holt-Winters 乘法趋势 + 乘法季节加法乘法
\[ \begin{aligned} \text{水平：} & \ell_t = \alpha y_t + (1-\alpha)(\ell_{t-1} + b_{t-1}) \\ \text{趋势：} & b_t = \beta(\ell_t - \ell_{t-1}) + (1-\beta)b_{t-1} \\ \text{季节：} & s_t = \gamma(y_t - \ell_{t-1} - b_{t-1}) + (1-\gamma)s_{t-m} \end{aligned} \]

11.5A VAR：向量自回归

当多个时间序列互相影响时

ARIMA 只能建模单个时间序列。但竞赛中经常遇到多个相关的序列——例如 GDP、利率、通胀三者互相影响。VAR (Vector AutoRegression) 是 AR 模型的多变量推广：每个变量不仅依赖自己的过去值，还依赖其他变量的过去值。

\[ \mathbf{y}_t = \mathbf{c} + \mathbf{\Phi}_1 \mathbf{y}_{t-1} + \mathbf{\Phi}_2 \mathbf{y}_{t-2} + \cdots + \mathbf{\Phi}_p \mathbf{y}_{t-p} + \boldsymbol{\varepsilon}_t \]

其中 \(\mathbf{y}_t\) 是 \(k \times 1\) 向量（k 个变量），\(\mathbf{\Phi}_i\) 是 \(k \times k\) 系数矩阵，\(\boldsymbol{\varepsilon}_t \sim N(0, \mathbf{\Sigma})\)。

VAR 的关键分析工具

脉冲响应函数 (IRF)：给变量 j 一个单位冲击 → 观察变量 i 在后续各期的响应。IRF 揭示了变量之间的动态传导机制。
方差分解 (FEVD)：变量 i 的预测误差方差中，有多大比例来自变量 j 的冲击？——量化了各变量之间的「解释力」。
滞后阶数选择：用 AIC、BIC、HQ 选择最优 p。VAR(p) 的参数量 = \(k + k^2 \times p\)——随 k 和 p 增长极快，需要足够的样本量。

竞赛技巧：VAR 适用于宏观经济、金融、环境等多变量时序问题。论文中：「我们建立 VAR(2) 模型分析 GDP、CPI 和 M2 之间的动态关系。脉冲响应分析表明，M2 的一个正冲击在 3 个季度后对 CPI 产生约 0.3% 的正向影响，持续约 6 个季度。」——这种分析在竞赛中非常出彩。但注意：VAR 要求所有序列都平稳——非平稳序列需要先差分或使用 VECM（见 11.5B）。

11.5B 协整与误差修正模型 (ECM)

当多个非平稳序列有「长期均衡关系」

两个非平稳序列（如 GDP 和消费）各自都是 I(1)，但它们的某个线性组合可能是 I(0)（平稳）——这就是协整 (Cointegration)。协整意味着两个序列之间存在长期均衡关系——它们不会「漂移」得太远。

Engle-Granger 两步法

Step 1 — 协整回归：\(y_t = \beta_0 + \beta_1 x_t + u_t\)（OLS 估计长期关系）。
Step 2 — 残差检验：对残差 \(\hat{u}_t\) 做 ADF 检验（使用专门的临界值，比标准 ADF 更严格）。如果残差平稳 → 存在协整。
Step 3 — 误差修正模型 (ECM)：

\[ \Delta y_t = \alpha (y_{t-1} - \beta_0 - \beta_1 x_{t-1}) + \gamma \Delta x_t + \varepsilon_t \]

其中 \(\alpha\) 是误差修正系数。由平稳性条件 \(|1+\alpha| < 1\) 得 \(-2 < \alpha < 0\)。其中 \(-1 < \alpha < 0\) 对应单调回归均衡，\(-2 < \alpha < -1\) 对应振荡式回归均衡。\(y_{t-1} - \beta_0 - \beta_1 x_{t-1}\) 是误差修正项——上一期的偏离量。在多变量 VECM 中不能给出统一的单个区间，具体稳定性应根据系统特征根判断。

竞赛技巧：协整 + ECM 是竞赛中处理多变量非平稳序列的标准方法。论文中：「Johansen 协整检验表明，GDP 和电力消费之间存在一个协整关系 (Trace Statistic = 18.3, p = 0.017)。ECM 估计显示，误差修正系数 α = -0.23——意味着偏离长期均衡后，每期约 23% 的偏离被修正。短期来看，电力消费增长 1% 伴随 GDP 增长 0.4%。」
Johansen 检验 vs Engle-Granger：Johansen 检验可以同时检验多个协整关系，且不预设哪个变量是「因变量」——是竞赛中更推荐的方法。

11.5C 格兰杰因果检验

「预测」意义上的因果，不是哲学意义上的因果

格兰杰因果 (Granger Causality) 的定义：如果加入 X 的过去值能显著改善对 Y 的预测（在已经包含 Y 的过去值的基础上），则 X 是 Y 的格兰杰原因。注意：这不是真正的因果关系——它只是说 X 的过去值对预测 Y 有帮助。

\[ \text{无约束模型：} \quad y_t = \alpha_0 + \sum_{i=1}^{p} \alpha_i y_{t-i} + \sum_{i=1}^{p} \beta_i x_{t-i} + \varepsilon_t \] \[ \text{约束模型：} \quad y_t = \alpha_0 + \sum_{i=1}^{p} \alpha_i y_{t-i} + \varepsilon_t \]

检验 \(H_0: \beta_1 = \beta_2 = \cdots = \beta_p = 0\)（X 不格兰杰导致 Y）。用 F 检验比较两个模型的 RSS。如果 p < 0.05 → 拒绝 H₀ → X 是 Y 的格兰杰原因。

关键警告：格兰杰因果 ≠ 真正因果！「公鸡打鸣」格兰杰导致「太阳升起」——因为打鸣在前，太阳升起在后。但这不意味着打鸣导致了太阳升起。在竞赛论文中，正确的表述是：「格兰杰因果检验表明，X 的过去值对预测 Y 具有显著的信息增量 (F = 4.32, p = 0.012)，但这一统计关系可能反映的是共同趋势或第三变量的影响，而非直接的因果关系。」

11.5D GARCH 模型

不只是预测值，还要预测波动

ARIMA 假设残差的方差是恒定的。但在金融和某些物理数据中，波动率本身会随时间变化——大的波动倾向于跟随大的波动，小的波动跟随小的波动（波动聚集效应，Volatility Clustering）。GARCH (Generalized AutoRegressive Conditional Heteroskedasticity) 专门建模这种时变方差。

\[ \begin{aligned} \text{GARCH(p,q)：} \quad & \sigma_t^2 = \omega + \sum_{i=1}^{q} \alpha_i \varepsilon_{t-i}^2 + \sum_{j=1}^{p} \beta_j \sigma_{t-j}^2 \\ \text{GARCH(1,1)：} \quad & \sigma_t^2 = \omega + \alpha \varepsilon_{t-1}^2 + \beta \sigma_{t-1}^2 \end{aligned} \]

GARCH(1,1) 的解释：\(\omega\) = 方差方程截距（常数项），\(\alpha\) = 昨日冲击对今日波动的影响（ARCH 效应），\(\beta\) = 昨日波动对今日波动的影响（GARCH 效应）。当 \(\alpha + \beta < 1\) 时过程平稳，无条件长期方差为 \(\sigma^2 = \omega/(1 - \alpha - \beta)\)。\(\alpha + \beta\) 越接近 1 → 波动持续越久。

竞赛技巧：GARCH 在金融建模竞赛中几乎是标配——股票收益率、汇率、比特币价格等的波动率预测。论文中：「Ljung-Box 检验显示收益率序列的平方存在显著自相关 (Q(20)=87.3, p<0.001)，表明存在 ARCH 效应。我们拟合 GARCH(1,1) 模型，α=0.12 (p=0.003)，β=0.83 (p<0.001)，α+β=0.95——表明波动冲击具有高度持续性，半衰期约为 ln(0.5)/ln(0.95) ≈ 13.5 天。」

11.5E KPSS 检验

ADF 的互补检验——双重确认平稳性

ADF 检验的原假设是「存在单位根（非平稳）」，备择假设是「平稳」。但 ADF 的检验功效较低——当数据接近单位根（φ ≈ 0.95）时，ADF 经常无法拒绝原假设。KPSS (Kwiatkowski-Phillips-Schmidt-Shin) 检验反转了原假设——H₀ 是「平稳」，H₁ 是「存在单位根」。

双重检验策略

同时使用 ADF 和 KPSS，四种可能的结果：

ADF 结果KPSS 结果结论
拒绝 H₀（平稳）不拒绝 H₀（平稳）序列平稳 ✓
不拒绝 H₀拒绝 H₀序列非平稳 → 差分
拒绝 H₀拒绝 H₀可能趋势平稳 → 去趋势
不拒绝 H₀不拒绝 H₀样本量不足，无法判断
竞赛技巧：竞赛论文中同时报告 ADF 和 KPSS 结果，体现严谨性。句式：「ADF 检验在 5% 显著性水平下无法拒绝单位根假设 (p = 0.23)，而 KPSS 检验在 5% 水平下拒绝平稳性假设 (p = 0.01)。两种检验一致表明序列是非平稳的，需要一阶差分。」这种双重确认在评审中会得到高度评价。

11.5F 变点检测 (Change Point Detection)

数据在什么时候「突然变了」？

时间序列的结构可能在某一点发生突变——均值突然上升、方差突然增大、趋势突然改变。变点检测 (Change Point / Structural Break Detection) 用于识别这些突变点。竞赛中常用于：政策干预效果评估、系统故障检测、市场拐点识别。

Chow 检验

Chow 检验检测已知的潜在变点（如政策实施日期）：

\[ F = \frac{(\text{RSS}_{\text{pooled}} - (\text{RSS}_1 + \text{RSS}_2)) / k}{(\text{RSS}_1 + \text{RSS}_2) / (n_1 + n_2 - 2k)} \sim F_{k, n_1+n_2-2k} \]

其中 RSS_pooled 是全部数据拟合一个模型的残差，RSS₁ 和 RSS₂ 是分段拟合的残差。如果 F 显著 → 存在结构性断点。

Bai-Perron 检验

当变点未知时，Bai-Perron 方法在所有可能的位置搜索多个变点，使用 BIC 或序贯检验确定最优变点数量和位置。

竞赛技巧：变点检测在竞赛中的应用场景：① 检测政策实施前后的结构性变化。② 疫情前后经济指标的变化点。③ 系统运行状态的突变（故障预警）。论文中：「Chow 检验表明，在 2020 年第 1 季度存在显著的结构性断点 (F=7.82, p<0.001)，与疫情爆发的时点一致。分段回归显示，疫情前趋势增长率为 3.2%/年，疫情后降为 0.8%/年。」

仪器 No.11 — 时间序列分解与 ACF/PACF

11.6 竞赛实战流程

时间序列竞赛流程

Step 1：可视化——画时序图，肉眼判断趋势和季节性。
Step 2：平稳性检验——ADF 检验 + KPSS 检验（双重确认）。
Step 3：差分——如果需要，确定差分阶数 \(d\) 和季节差分阶数 \(D\)。
Step 4：ACF/PACF → 初步确定 p, q, P, Q 的范围。
Step 5：网格搜索 + AICc → 选择最优模型。
Step 6：残差诊断 → Ljung-Box 检验（白噪声）、Q-Q 图（正态性）。
Step 7：预测 + 95% 置信区间 → 论文中最关键的图。
Step 8：模型对比 → 也可以跑一个 ETS 或朴素预测作为基线。
